# Privacy Policy — problem-consciousness

**This plugin does not collect, store, or transmit any personal data.**

- All skills are local instruction files (Markdown) read by Claude Code on your own machine.
- The only hook (`SessionStart`) prints a short static text reminder to your console — it sends nothing anywhere.
- The plugin has no telemetry, no analytics, no bundled network calls, and no external services of its own.

Any network access that occurs during your session is initiated by **you** through Claude Code and the tools you invoke — not by this plugin.
